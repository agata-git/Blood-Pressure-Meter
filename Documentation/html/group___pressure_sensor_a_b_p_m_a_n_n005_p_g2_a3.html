Detailed Description

Module used for initialization I2C1 bus used for communication with a piezoresistive silicon pressure sensor from Honeywell Basic Amplified ABP Series. Calibrated output values for pressure are updated at approximately 1 kHz for analog and 2 kHz for digital. According to Honeywell Technical note, Honeywell's digital output pressure sensors are designed to work as slaves and will therefore only respond to requests from a master device. Following the address and read bit from the master, Honeywell digital output pressure sensors are designed to output up to 4 bytes of data, depending on the sensor options and the needs of the application. Sensor used in project gives two data bytes that are the compensated pressure output, along with sensor status bits.
